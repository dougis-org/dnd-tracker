# Environment Variables Setup

**Objective:** Establish secure and consistent environment variable management for all environments.

**Tasks:**

- [ ] Define required environment variables for local, staging, and production
- [ ] Document variable purpose and usage
- [ ] Add .env.example file
- [ ] Ensure variables are loaded in Next.js and deployment

**Acceptance Criteria:**

- All required variables are documented
- .env.example exists and is up to date
- App loads variables correctly in all environments
