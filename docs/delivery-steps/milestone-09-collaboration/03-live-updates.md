# Live Combat Updates

**Objective:** Synchronize combat state and actions in real time for all users.

**Tasks:**

- [ ] Sync combat state across all users in campaign
- [ ] Broadcast actions and updates instantly
- [ ] Handle connection loss and reconnection
- [ ] Add UI for live update status
- [ ] Write tests for sync logic

**Acceptance Criteria:**

- Combat state is synchronized for all users
- Actions are broadcast instantly
- Connection loss is handled gracefully
- Sync logic is tested
